Caroline's Portfolio
A responsive personal portfolio website


This is a fully responsive portfolio website featuring modal-style content sections
and smooth animations. The site includes sections for introduction, work samples,
about information, and contact details. Built with modern web technologies and
optimized for all screen sizes.

The template foundation is based on the "Dimension" design by HTML5 UP, customized
and adapted for personal portfolio use.

Images and content should be replaced with your own materials.
