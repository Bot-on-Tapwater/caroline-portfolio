Caroline's Portfolio
A responsive personal portfolio website


This is a fully responsive portfolio website featuring smooth animations and 
modern design elements. The site includes sections for introduction, about 
information, work samples, and contact details. Built with modern web 
technologies and optimized for all screen sizes.

The template foundation is based on the "Luther" design by Styleshout, 
customized and adapted for personal portfolio use.

Images and content should be replaced with your own materials.
